# flowkore Website

Landing page para flowkore - Workflow Optimization para Retail de Alto Volumen en Latinoamérica.

## Descripción

Sitio web corporativo diseñado con metodología process-first, enfocado en comunicar la propuesta de valor de flowkore para empresas de retail que buscan optimizar sus procesos operativos.

## Estructura del Proyecto

```
flowkore-website/
├── index.html          # Página principal
├── styles.css          # Estilos principales
├── privacidad.html     # Política de privacidad
├── terminos.html       # Términos y condiciones
└── assets/             # Recursos (imágenes, etc.)
```

## Tecnologías

- HTML5
- CSS3 (Variables CSS, Flexbox, Grid)
- JavaScript vanilla
- Google Fonts (Inter)

## Características

- Diseño responsivo
- Animaciones CSS optimizadas
- Accesibilidad (ARIA labels, contraste de colores)
- SEO optimizado (meta tags, Open Graph, Twitter Cards)
- Performance optimizada (fonts preconnect, CSS externo)

## Deployment

El sitio está configurado para deployment en Vercel. El directorio `.vercel` contiene la configuración del proyecto.

## Desarrollo Local

Simplemente abre `index.html` en tu navegador o usa un servidor local:

```bash
# Con Python
python3 -m http.server 8000

# Con Node.js
npx serve
```
